# 📱 FRAS: Mobile Application

The mobile companion app for the Fleet Reporting & Analytics System, built with React Native and Expo. 

---

## 🚀 Key Features

- **📊 Mobile Dashboard**: Quick-access KPI cards for on-the-go fleet monitoring.
- **📑 Report Viewer**: View and download reports directly to your mobile device.
- **🔐 Secure Access**: Integrated biometric authentication and secure JWT handling.
- **📩 Sharing**: Integrated system sharing for PDF/Excel reports.

---

## 🛠️ Technology Stack

- **React Native / Expo**: Core cross-platform engine.
- **React Navigation**: Seamless screen orchestration.
- **Axios**: API integration with optimized network handling.
- **Async Storage / Secure Store**: Local data and session management.
- **React-Native-Chart-Kit**: Mobile-optimized data visualization.

---

## 🚦 Getting Started

### 1. Prerequisites
Ensure you have the [Expo Go](https://expo.dev/expo-go) app installed on your smartphone.

### 2. Installation
```bash
npm install
```

### 3. Development Server
```bash
npx expo start
```

### 4. Direct Device Access
Scan the QR code printed in the terminal with your phone's camera (iOS) or Expo Go app (Android).
